## What is a journey?

A **journey** is a learning path made of chapters. Each chapter is usually a video (YouTube link). Journeys can be private or public.

## Creating a journey

### From a YouTube playlist

1. On **Home**, click **Create New Journey**
2. Select **Import from YouTube playlist**
3. Paste the playlist URL (e.g. `https://www.youtube.com/playlist?list=PL...`)
4. Choose **Public** if you want others to discover it
5. Click **Create from playlist**

All videos are imported as chapters. Titles come from the playlist.

### Custom journey

1. On **Home**, click **Create New Journey**
2. Select **Create Custom Journey**
3. Fill in **Journey name** and **Description**
4. Choose **Public** if you want to share
5. Click **Create**

You’ll need to add chapters manually (see [Adding chapters](#adding-chapters)).

## Managing a journey

### Open a journey

On Home, click any journey row to open it. You’ll see:

- Journey title and description
- Visibility (Public / Private)
- Progress %
- **Notes** link
- **Chapters** table

### Add a chapter

1. Open the journey
2. Click **+ Add New Chapter**
3. Enter:
   - **Title** — Chapter name
   - **Description** (optional)
   - **Video URL** — YouTube video link
4. Save

Chapters are ordered by number. New ones are appended.

### Edit a chapter

1. Open the journey
2. Click **Edit** next to the chapter
3. Update title, description, or video URL
4. Save

### Delete a chapter

1. Open the journey
2. Click **Delete** next to the chapter
3. Confirm deletion

### Mark chapter complete

Use the **Mark as complete** checkbox to track progress. Journey progress updates automatically.

## Deleting a journey

1. On **Home**, find the journey
2. Click **Delete** in the Actions column
3. Confirm

Deleting a journey removes its chapters and notes.

## Visibility

- **Public** — Visible in **Explore**. Others can browse and fork it.
- **Private** — Only you can see it in your Home.

You can change visibility when creating or editing a journey.
